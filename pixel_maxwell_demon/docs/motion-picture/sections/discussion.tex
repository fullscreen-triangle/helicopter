\subsection{Practical Limitations}

\subsubsection{Computational Overhead}

Entropy computation requires $O(N \log N)$ operations per frame (optical flow dominates). For 4K video (8.3M pixels @ 60 fps):
\begin{itemize}
\item Real-time encoding: requires 498 GFLOPS (challenging for mobile devices)
\item Playback: real-time with pre-computed entropy index
\end{itemize}

\textbf{Solution}: Hardware acceleration (GPU/FPGA) or compressed-domain entropy computation.

\subsubsection{Back Face Quality Degradation}

Conjugate frame generation is imperfect. While SSIM > 0.92, perceptual studies show 58\% detection rate at 50\% back face content. This limits practical usage to:
\begin{itemize}
\item Brief backward scrubs (<1 second)
\item Applications where perfect fidelity is not critical
\item Scenarios where back face represents valid alternative (scientific visualization)
\end{itemize}

\textbf{Improvement}: Generative models (GANs, diffusion models) could produce higher-quality conjugate frames while maintaining entropy constraints.
